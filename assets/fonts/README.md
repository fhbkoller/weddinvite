# Custom Fonts - Wedding Invitation

This directory contains the custom font used in the project:
- `Joliet-Regular.ttf`
- `Joliet-Regular.woff2`

This font is primarily used for the main aesthetic text (e.g., elements using the `font-esthetic` and `font-names` classes).

The project also uses fonts provided by **Google Fonts**, which are imported directly into the main HTML file (`index.html`):
- **DM Sans**
- **Libre Caslon Text**
